Lineamientos para el mantenimiento de repositorios   
########################################################  

1. GenomicConservationLab-ECOSUR será la responsable de generar nuevos repositorios y asignar el rol de acceso a cada miembro del equipo.

2. La cuenta GenomicConservationLab-ECOSUR del laboratorio será administrada por @acanedo92

3. Para crear nuevos reporitorios en la cuenta deberán escribir una solicitud a través del formulario: https://forms.office.com/r/UDaDAUF375 ​también puedes escanear el cógido QR que encontrarás en este mismo repositorio.

4. Para configurar Git deberás seguir las instrucciones que se encuentran en el archivo:
GitConfig.R # Para usar a través de Rsutio
GitConfigLinux.sh # Para usar a través de la terminal de linux

5. Cada usuario podrá clonar el repositorio desde su terminal o desde Rstudio. 

6. Nunca trabajar en la rama principal (origin main). Para trabajar sus propios cambios crear una nueva rama. Ver las instrucciones en el archivo Git_CommandLine.sh. Ej: $ git checkup -b newBranch 

7. Los lineamientos para subir scripts se encuentran en el archivo: ScripsRules.sh

8. Aquí puedes consultar el tutorial para utilizar git: https://happygitwithr.com/
